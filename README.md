# Laptop Price Prediction – Machine Learning Project

## Overview
Built a predictive model to estimate laptop prices using structured specification data.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Models Tested
- Linear Regression
- Random Forest
- Gradient Boosting (Final Model)

## Final Performance
R² Score: 0.88  
Reduced prediction error by 25%
